# zerolength 예제 실행 방법

이 프로젝트는 Vite와 React를 사용합니다. 다음 단계로 개발 서버를 실행할 수 있습니다.

1. 의존성 설치

```bash
npm install
```

2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 주소로 접속하면 애플리케이션을 확인할 수 있습니다.

3. 빌드 및 배포 파일 생성

```bash
npm run build
```

`dist` 폴더에 정적 파일이 생성되며, `npm run serve`로 빌드 결과를 미리 볼 수 있습니다.
